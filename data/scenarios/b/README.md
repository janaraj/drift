# data/scenarios/b/

Hand-authored scenarios for **B — persona / behavioral violation**.

Each scenario is a YAML file conforming to the common schema in
`src/drift/scenarios/schema.py`, with `behavior_config` carrying the
assigned persona, forbidden tokens, and which of the three sub-behaviors
(human-claim / persona-name violation / self-contradiction) it targets.

Populated by Unit 7.3.
