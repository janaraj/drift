# configs/

YAML configuration files for the project. Populated by later units.

```
configs/
  behaviors/{e,b}.yaml      # behavior config: judge cls, scenario dir, reward weights — Unit 2.5 + 7.4
  training/{sft,grpo}_*.yaml  # per-run training configs — Unit 4.2 onward
  eval/*.yaml               # per-sweep eval configs — Unit 3.5 onward
```
